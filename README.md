# Memories App

A Simple Memories sharing app built on MERN Stack.

## Demo

Click [here](https://link) for demo.

## Installation

```bash
$ npm install
$ npm run client-install
```

## Running the app

```bash
# Run server & client in development
$ npm run dev

# Run server only
$ npm start

# Run client onky
$ npm run client
```
